# 📖 About

Discord Zapret YouTube is a modern Windows build based on **zapret**, providing ready-to-use DPI bypass strategies, diagnostics and automatic configuration.

Designed for users who want to get started in just a few clicks without manual WinDivert configuration.

# ❓ FAQ

<details>

<summary><b>Nothing happens after launch?</b></summary>

Check your antivirus.

Verify WinDivert wasn't removed.

</details>

<details>

Update IPSet.

</details>

# 🛡 Security

> [!IMPORTANT]

This project uses **WinDivert**.

Some antivirus software may detect it as:

- RiskTool
- PUA
- WinDivert

This is expected behavior and **does not indicate malware**.
